Github + Hugo

一隻鹹魚 · May 30, 2020

建立步驟摘要

  1. 申請Github帳號，建立網站的repository。
  2. 到Hugo網站選擇主題(theme)，下載至PC，建立本地(local)端測試環境。
  3. 安裝Visual Studio Code，修改config.toml進行主題客製化設定。
  4. 啟動Hugo Server，於本地端檢視客製後之網站顯示是否正常。
  5. 使用Hugo.exe產生實體靜態網頁於publish目錄下。
  6. 使用GitHub Desktop軟體，讓本地端網頁與Github的repository同步，完成佈署。
